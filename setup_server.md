# Setting up a server

## Checklist

- setup basic functionality in the server
  - create a non-root user
  - setup password-less login for the root
  - setup password-less login for the user
  - install basic packages
    - neovim
    - curl
    - node.js
    - bun
    - golang
    - docker
  - setup neovim configs
  - setup firewall
